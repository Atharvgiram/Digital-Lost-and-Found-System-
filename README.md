# Digital-Lost-and-Found-System-
The Digital Lost and Found System is a web-based application developed to digitize the process of reporting, managing, and tracking lost and found items within an organization or campus. The system eliminates manual registers by providing a centralized database-driven solution.
To create a secure and structured platform that enables users to report lost items, view found items, and track claim status efficiently.
Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Server: Apache (XAMPP)
